# 项目概述

这是一个用于学习 Spring AI 的**多模块 Maven 项目**，展示了各种 AI 能力，包括对话、多模态交互、函数调用、MCP 集成、RAG 管道和智能体工作流。

## 技术栈

- **Spring Boot**: 3.5.8
- **Spring AI**: 1.1.0-M4
- **Java**: 21
- **构建工具**: Maven

## 项目目标

展示 Spring AI 的核心能力：
- 对话和聊天
- 多模态交互（文本 + 图片）
- 函数调用和工具使用
- MCP（模型上下文协议）集成
- RAG（检索增强生成）管道
- 智能体工作流
- 对话记忆管理

## 学习路径

1. **spring-ai-example** - 从基础开始，学习核心功能
2. **mcp-weather-server** - 理解 MCP 工具集成
3. **rag-etl-core** - 学习 RAG 文档处理抽象
4. **rag-etl-opensearch** - 实现 RAG 向量存储
5. **spring-ai-alibaba-graph** - 探索智能体工作流
